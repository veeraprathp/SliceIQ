                   TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Problem: "Failed to generate quiz"
→ Check proxy is running (/c/Python314/python proxy.py)
→ Check web server running (/c/Python314/python -m http.server 8000)
→ Check http://localhost:8001 responds

Problem: "Hand tracking not working"
→ Allow camera permission in browser
→ Ensure good lighting
→ Move hand clearly in front of camera

Problem: "No cards spawning"
→ Open browser console (F12)
→ Check for JavaScript errors
→ Refresh page (Ctrl+R)
